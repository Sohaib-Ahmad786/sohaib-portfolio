## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/Sohaib-Ahmad786/sohaib-portfolio.git
```

Go to project directory:

```bash
cd sohaib-portfolio
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
